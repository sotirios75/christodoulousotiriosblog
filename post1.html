Οδηγός για το Web Scraping με Python

24 Αυγούστου, 2025 Τεχνολογία Χριστοδούλου Σωτήριος

Το web scraping είναι μια ισχυρή τεχνική για τη συλλογή δεδομένων από ιστοσελίδες. Στον οδηγό αυτό, θα δούμε πώς μπορούμε να χρησιμοποιήσουμε τη γλώσσα προγραμματισμού Python και τις δημοφιλείς βιβλιοθήκες Beautiful Soup και Requests για να αυτοματοποιήσουμε τη διαδικασία.

Τι είναι το Web Scraping;

Με απλά λόγια, το web scraping είναι η διαδικασία εξαγωγής πληροφοριών από ιστοσελίδες. Αντί να αντιγράφουμε και να επικολλάμε τα δεδομένα με το χέρι, μπορούμε να γράψουμε ένα πρόγραμμα που θα κάνει αυτή τη δουλειά για εμάς, εξοικονομώντας μας χρόνο και κόπο.

Βήμα 1: Εγκατάσταση των Απαραίτητων Βιβλιοθηκών

Για να ξεκινήσουμε, πρέπει να εγκαταστήσουμε τις βιβλιοθήκες Requests (για τη λήψη του HTML κώδικα) και Beautiful Soup (για την ανάλυση του HTML). Ανοίξτε το τερματικό σας και πληκτρολογήστε:

pip install requests beautifulsoup4

Βήμα 2: Ο Κώδικας

Ας δούμε ένα παράδειγμα όπου θα συλλέξουμε τους τίτλους των άρθρων από ένα blog:


import requests
from bs4 import BeautifulSoup

url = 'https://www.example.com/blog'  # Αντικαταστήστε με τη σωστή URL
response = requests.get(url)

if response.status_code == 200:
    soup = BeautifulSoup(response.text, 'html.parser')
    article_titles = soup.find_all('h2')  # Εντοπίστε τους σωστούς HTML selectors

    for title in article_titles:
        print(title.text.strip())
else:
    print(f"Αποτυχία λήψης της σελίδας. Status Code: {response.status_code}")
